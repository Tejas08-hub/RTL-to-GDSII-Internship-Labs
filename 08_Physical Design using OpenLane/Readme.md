# Physical Design using OpenLane

## Overview

This repository contains a complete hands-on implementation of the **RTL-to-GDSII Physical Design Flow** using the **OpenLane** open-source ASIC design flow. The projects demonstrate every major stage involved in transforming an RTL design into a fabrication-ready GDSII layout using the **Sky130A Process Design Kit (PDK)**.

The repository includes practical implementations of two example designs:

- Decoder
- Serial Parallel Multiplier (SPM)

Each lab focuses on a specific phase of the physical design flow, providing source files, configuration files, layout databases, reports, logs, screenshots, and detailed documentation.

---

# Objectives

- Understand the complete RTL-to-GDSII flow.
- Learn ASIC physical design using OpenLane.
- Explore each stage of the OpenROAD-based implementation flow.
- Generate fabrication-ready GDSII layouts.
- Perform signoff verification using DRC and LVS.

# RTL-to-GDSII Flow

The physical implementation follows the standard ASIC design flow:

```
RTL Design
     │
     ▼
Logic Synthesis
     │
     ▼
Floorplanning
     │
     ▼
Placement
     │
     ▼
Clock Tree Synthesis (CTS)
     │
     ▼
Routing
     │
     ▼
Design Rule Check (DRC)
     │
     ▼
Layout Versus Schematic (LVS)
     │
     ▼
GDSII Generation
```
